# 🌤️ Previsão do Tempo

Aplicação web que exibe as condições climáticas atuais de qualquer cidade do mundo, consumindo dados em tempo real de uma API de meteorologia.

## ✨ Funcionalidades

- 🔍 Busca de clima por nome de cidade
- 🌡️ Exibe temperatura atual, sensação térmica e condição do tempo
- 💧 Informações de umidade e vento
- 📱 Layout responsivo para mobile e desktop

---

## 🛠️ Tecnologias utilizadas

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura da aplicação |
| CSS3 | Estilização e responsividade |
| JavaScript (ES6+) | Lógica, consumo de API e manipulação do DOM |
| API de Clima | Dados meteorológicos em tempo real |

---

## ▶️ Como rodar o projeto

```bash
# 1. Clone o repositório
git clone https://github.com/brunoluiz131094/Previs-o-do-Tempo.git

# 2. Acesse a pasta
cd Previs-o-do-Tempo

# 3. Abra o arquivo index.html no navegador
# (Basta dar dois cliques no arquivo ou usar a extensão Live Server do VS Code)
```

> ⚠️ Para funcionar, é necessário ter uma chave de API válida (ex: [OpenWeatherMap](https://openweathermap.org/api)). Insira sua chave no arquivo `script.js`.

---

## 📚 O que aprendi neste projeto

- Consumo de API REST com `fetch` e `async/await`
- Manipulação dinâmica do DOM com JavaScript
- Tratamento de erros e feedback ao usuário
- Organização de código em funções reutilizáveis
